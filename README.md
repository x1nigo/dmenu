# x1nigo's build of dmenu
My build of the suckless, dynamic menu.

## Patches
- `alpha` (transparency)
- `caseinsensitive` (looks for matches, regardless of case sensitivity)
- `mousesupport` (adds selection via mouse)

## Requirements
- `libxft-bgra` - You need this if you want to render colored emojis in my build of dmenu. Otherwise, you don't need this.

## References
- For the original version, see http://tools.suckless.org/dmenu/.
