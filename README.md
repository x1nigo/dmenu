# x1nigo's build of dmenu
My build of the suckless, dynamic menu.

## Patches
- alpha
- caseinsensitive
- fuzzyhighlight
- fuzzymatch
- grid
- gridnav
- lineheight
- listfullwidth

## Requirements
- ***libxft-bgra*** - You need this if you want to render colored emojis in my build of dmenu. Otherwise, you don't need this.
